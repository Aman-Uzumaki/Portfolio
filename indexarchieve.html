AatmaNirbhar Yuva (ANY)

Atmanirbhar Yuva, abbreviated as ANY, is an innovative web platform strategically designed to empower students residing in rural areas through comprehensive educational support. This initiative aims to bridge the educational gap by providing a technological solution tailored for economically disadvantaged students in rural communities.

Smart Grade Calculator

This project utilizes Java and MySQL to create a dynamic grade generator. The system empowers teachers to design exam structures, including the number of exams and subjects. Teachers can enter individual student marks, and students can log in to view their results. The project showcases Java and MySQL connectivity and allows flexibility for each teacher to determine the subjects and exams they conduct.
